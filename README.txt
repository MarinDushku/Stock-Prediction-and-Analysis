# Stock Market Trend Analysis Dashboard

## Project Overview
A comprehensive, interactive dashboard for stock market analysis and forecasting using Python and Streamlit.

## Features
- Real-time stock data retrieval
- Technical indicator calculations
- Price forecasting using Prophet
- Interactive visualizations
- Multiple prediction horizons

## Prerequisites
- Python 3.8+
- pip
